# Shortcuts and Macros

The map editor supports shortcuts for most functions such as, as well as editing macros such as copying and pasting.

**Space:** Add object

**Delete:** Delete object

**LeftControl:** Multiselect

**Escape:** Deselect

**C:** Change Gizmo

**G:** Toggle Snap

**Control + C:** Copy

**Control + V:** Paste

**Control + X:** Cut

**Control + Z:** Undo

**Control + Y:** Redo

**Control + S:** Save
